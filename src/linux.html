I use (K)ubuntu and Archlinux.

Kubuntu

Kubuntu combines KDE's great desktop with ubuntu's software repos. Ubuntu comes with a lot of things pre-configured(somehow not TLP on laptops? no clue why) and is great for new linux users. It is stable, and I like it. Comes way too bloated OOTB though.

I use a KWin rule to hide all titlebars and using Krohnkite(newer, maintained fork Bismuth) to automatically tile leaves you with a tiling window manager, that you can disable when you need to. Most things just werk, and it is reliable

Archlinux/Blackarch

Archlinux gives you the power to tweak the system as you wish. You must set up EVERYTHING. It IS a timesink OS. No doubt about it.  The control does leave you with something rewarding-a system you know every nook and cranny of. If anything goes wrong-you know how to fix it. It is a great distribution for those who want to get deeper into Linux. The archlinux installation I have is very small in disk footprint, has almost no software, and I use it when I'm not on wallpower, consuming 7-9W(very good for a ""gaming"" laptop!)

For security tools, I use blackarch linux, basically Archlinux with the blackarch repos. Theres thousands  of tools available. Of course you aren't going to use them all, but it is really nice to have all the important tools already present on your system. For some weird reason the "Slim" edition ISO does not have the TWM configurations, instead you are made to use the xfce environment with their tacky theming. You can curl/git clone the configs from their github.
